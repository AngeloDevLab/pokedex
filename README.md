# Pokédex

A small practice project built to learn how to work with APIs in JavaScript.

## Overview

This project uses the PokéAPI to fetch and display Pokémon data in a simple web interface.

## Features

* Fetch Pokémon data from an API
* Display Pokémon names and images
* Dynamic rendering with JavaScript
* Basic project structure with separated logic (API / UI)

## Tech Stack

* HTML
* CSS
* JavaScript (Vanilla)
* PokéAPI

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser

## API

https://pokeapi.co/

## Learning Goals

* Understanding API requests with `fetch`
* Working with `async / await`
* Handling and displaying JSON data
* Structuring JavaScript code
